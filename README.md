# The Bixt - Smoky

A newsletter website following The Silver Spooners—a group of friends navigating love, life, and London. Rituals, Reality, and Recipes.

## About

Smoky is a newsletter that weaves together stories of friendship, quarter-life uncertainty, and the small rituals that keep us grounded. Each episode features heartfelt conversations, Sunday morning scrambled eggs, and the courage it takes to change.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Fonts**: Inter, Cormorant Garamond, Reenie Beanie
- **Deployment**: Vercel

## Features

- Episode pages with rich typography and animations
- Photo gallery with taped polaroid effect
- Light/dark theme transitions
- WhatsApp subscription integration
- Responsive design

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Project Structure

```
src/
├── app/           # Next.js app router pages
├── components/    # React components
└── data.ts        # Episode content and site config
```
